
# Health Tracker Website

A prototype **health tracking web application** developed during a hackathon to help users monitor their health habits, nutrition, and fitness activities.

The platform provides simple tools for tracking calories, exploring nutrition information, and learning about healthy lifestyle practices.

---

## Project Overview

The goal of this project was to create a basic health-focused website where users can:

* Track calorie intake
* Learn about nutrition
* Explore fitness-related information
* Access simple health tools

This project was developed in a **limited time during a hackathon**, so the main focus was building a functional prototype rather than a production-ready system.

---

## Features

• Calorie tracking interface
• Nutrition information pages
• Fitness guidance section
• Health awareness content
• Basic login page
• Multiple informational pages related to health and diet

---

## Tech Stack

Frontend

* HTML5
* CSS3
* JavaScript

Additional scripts

* Python (for basic calculations)
